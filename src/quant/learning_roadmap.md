## 阶段一：金融基础

### 核心目标

理解金融市场运作规则，建立正确的市场认知。

### 1.1 市场基础知识

**必备概念**：

| 概念 | 说明 | 算法视角理解 |
|------|------|-------------|
| 交易所 | 上交所、深交所、北交所 | 市场的"服务器" |
| 交易时段 | 9:30-11:30, 13:00-15:00 | 数据采样时间窗口 |
| T+1制度 | 当日买入次日才能卖出 | 序列中的延迟约束 |
| 涨跌停 | ±10% (20%创业板/科创板) | 价格变化的上限约束 |
| 最小变动价 | 0.01元 | 价格的离散化 |

**思考题**：
1. 为什么要有T+1制度？
2. 涨跌停对策略设计有什么影响？
3. 如何处理涨跌停导致的"买不进/卖不出"？

### 1.2 交易品种对比

| 品种 | 代码规则 | 交易制度 | 杠杆 | 适合策略类型 |
|------|----------|----------|------|-------------|
| 主板股票 | 600xxx, 000xxx | T+1 | 1:1 | 趋势、价值 |
| 创业板 | 300xxx | T+1 | 1:1 | 成长、动量 |
| 科创板 | 688xxx | T+1 | 1:1 | 硬科技主题 |
| ETF | 51xxxx, 159xxx | T+1 | 1:1 | 行业轮动、配置 |
| LOF | 16xxxx | T+1 | 1:1 | 折价套利 |
| 期货 | 对应合约 | T+0 | 8-15倍 | 日内、趋势 |
| 期权 | 对应合约 | T+0 | 变动 | 波动率、对冲 |
| 可转债 | 12xxxx | T+0 | 1:1 | 转股套利 |

**入门建议**：先从A股股票/ETF开始，熟悉规则后再考虑其他品种。

### 1.3 订单类型

```
市价单  ──▶  以最优价格立即成交（可能滑点）
限价单  ──▶  指定价格，挂单等待
止损单  ──▶  触发条件后转为市价单
条件单  ──▶  满足条件时自动下单
冰山单  ──▶  隐藏大单，分批成交
```

**量化为什么关心订单类型**？
- 回测时需要模拟真实成交
- 好策略要考虑"滑点"成本
- 大资金需要考虑"冲击成本"

### 1.4 交易成本详解

**A股交易成本构成**：

```
买入成本 = 印花税(0) + 佣金(万2.5~万5) + 过户费(万0.1)
卖出成本 = 印花税(千1) + 佣金 + 过户费

完整来回 ≈ 0.2% ~ 0.3% (买卖各一次)
```

**为什么重要**？
- 日内交易：来回0.3%，很难盈利
- 高频交易：成本是最大敌人
- 回测必须计入成本，否则是"纸上富贵"

**滑点**：
- 期望价格 vs 实际成交价的差异
- 流动性越差，滑点越大
- 大单会产生"冲击成本"

- [ ] 理解T+1/T+0的区别
- [ ] 知道至少3种金融产品特性

## 阶段二：量化分析核心

### 核心目标

掌握量化分析的核心技术：数据、指标、统计方法。

### 2.1 数据获取

**常见数据源**：

| 数据源 | 类型 | 覆盖范围 | 成本 |
|--------|------|----------|------|
| Tushare | 接口 | A股/港股/美股 | 免费有限额 |
| Akshare | 接口 | A股为主 | 免费 |
| Baostore | 本地 | A股历史 | 免费 |
| 聚宽/米筐 | 平台 | A股全 | 免费试用/付费 |
| Wind/Choice | 终端 | 全市场 | 昂贵 |

**数据类型**：
```
日线数据 ──▶  开高低复权价、成交量、成交额
分钟数据  ──▶  1min/5min/15min/30min/60min
tick数据  ──▶  逐笔成交（需要高频/专业数据）
财务数据  ──▶  ROE、净利润、营收等
基本面    ──▶  行业分类、估值指标
```

**数据质量问题**：
- 停牌处理
- 复权处理（前复权/后复权）
- ST股票处理
- 上市前数据剔除

### 2.2 技术指标体系

**趋势类指标**：
- **MA (移动平均)**：SMA、EMA、WMA
  - 用途：判断趋势方向、支撑阻力
- **MACD**：快线-慢线
  - 用途：动量变化、趋势确认
- **BOLL (布林带)**：均值 ± N倍标准差
  - 用途：波动率、超买超卖
- **ATR**：真实波幅
  - 用途：止损设置、波动率度量

**动量类指标**：
- **RSI**：相对强弱指标
  - 用途：超买超卖判断
- **KDJ**：随机指标
  - 用途：短线买卖信号
- **ROC**：变动率
  - 用途：动量强度

**成交量类指标**：
- **OBV**：能量潮
- **VWAP**：成交量加权平均价
- **量比**：当前成交量/平均成交量

**学习重点**：
1. 每个指标的计算公式
2. 指标背后的逻辑
3. 指标的适用场景
4. 指标的局限性

### 2.3 K线形态

**基础形态**：
```
阳线/阴线  ──▶  收盘价>开盘价 / 收盘价<开盘价
十字星    ──▶  收盘价≈开盘价（犹豫）
锤子线    ──▶  长下影+小实体（可能反转）
流星线    ──▶  长上影+小实体（可能反转）
大阳/大阴 ──▶  强势方向
```

**组合形态**：
- 双底/双顶
- 头肩顶/头肩底
- 上升/下降三角形
- 旗形整理

**算法视角**：K线形态 = 时间序列的模式识别问题

### 2.4 统计分析方法

**分布分析**：
- 收益率分布（通常不是正态分布）
- 偏度与峰度（肥尾效应）
- QQ图检验正态性

**平稳性检验**：
- ADF检验 (Augmented Dickey-Fuller)
- 价格序列通常不平稳
- 收益率序列通常平稳
- 协整关系（长期均衡关系）

**相关性分析**：
- 相关系数矩阵
- 滚动相关（时变关系）
- 因子相关性检验

✅ 完成标志：
- [ ] 能独立获取历史行情数据
- [ ] 手动实现5+技术指标
- [ ] 理解复权的重要性
- [ ] 知道ADF检验的意义

## 阶段三：策略开发实战

### 核心目标

从理论到实践，开发可运行的策略。

### 3.1 策略分类

```
┌─────────────────────────────────────────┐
│              量化策略分类               │
├─────────────────────────────────────────┤
│                                         │
│  按持仓时间：                            │
│  ├── 高频 (毫秒-秒级)                   │
│  ├── 日内 (分钟级)                      │
│  ├── 波段 (日-周级)  ← 建议从这里开始   │
│  └── 长线 (月-年级)                     │
│                                         │
│  按逻辑类型：                            │
│  ├── 趋势跟踪                           │
│  ├── 均值回归                           │
│  ├── 动量策略                           │
│  ├── 统计套利                           │
│  └── 机器学习                           │
│                                         │
└─────────────────────────────────────────┘
```

### 3.2 经典策略一：趋势跟踪

**核心思想**：强者恒强，追涨杀跌

**代表策略**：

1. **Dual Thrust**
   - 逻辑：利用昨日波幅确定上下轨
   - 优点：简单有效、适合趋势
   - 缺点：震荡市频繁止损

2. **海龟交易法则**
   - 逻辑：20日突破入场
   - 优点：经典趋势策略
   - 缺点：需要大资金、回撤大

3. **MACD策略**
   - 逻辑：金叉买入、死叉卖出
   - 优点：经典、易懂
   - 缺点：滞后严重

**趋势策略特点**：
- 胜率不高（30-50%）
- 盈亏比高（大赚小亏）
- 回撤期间难熬

### 3.3 经典策略二：均值回归

**核心思想**：物极必反，低买高卖

**代表策略**：

1. **布林带回归**
   - 逻辑：价格触及下轨买入、触及上轨卖出
   - 优点：逻辑简单
   - 缺点：强趋势被套

2. **RSI策略**
   - 逻辑：RSI<30超卖出，RSI>70超买卖
   - 优点：反应快
   - 缺点：钝化问题

3. **配对交易**
   - 逻辑：买入相对弱的、做空相对强的
   - 优点：市场中性
   - 缺点：需要协整检验

**均值回归特点**：
- 胜率较高（50-70%）
- 盈亏比较低
- 依赖波动率

### 3.4 经典策略三：动量策略

**核心思想**：过去强者继续强

**动量因子**：
- ROC（过去N日涨幅）
- 相对强度（相对大盘/行业）
- 成交量动量

**动量 vs 趋势**：
- 动量：横截面比较（选股）
- 趋势：时间序列比较（择时）

### 3.5 机器学习策略

**应用场景**：

| 场景 | 方法 | 特征 | 标签 |
|------|------|------|------|
| 价格预测 | LSTM/XGBoost | 历史价量、技术指标 | 未来收益率 |
| 方向判断 | 分类模型 | 技术指标、因子 | 涨/跌/平 |
| 模式识别 | CNN/图像模型 | K线图像 | 后续走势 |
| 仓位优化 | 强化学习 | 市场状态 | 最优仓位 |

**你的算法优势**：
- 特征工程经验
- 模型调参能力
- 过拟合防范意识
- 集成学习方法

**量化特有的ML问题**：
- 样本不平衡（涨跌比不是1:1）
- 数据非平稳（分布漂移）
- 特征信噪比低
- 过拟合风险高

## 阶段四：回测与风控

### 核心目标

正确评估策略，防范实盘风险。

### 4.1 回测原理

**什么是回测**：
```
历史数据 ──▶  应用策略 ──▶  模拟交易 ──▶  计算收益
```

**回测的陷阱**：

| 陷阱 | 说明 | 后果 |
|------|------|------|
| 未来函数 | 用了"未来"的数据 | 虚高收益 |
| 幸存者偏差 | 忽略退市股票 | 高估收益 |
| 前视偏差 | 知道历史选参数 | 过拟合 |
| 忽略成本 | 不算手续费/滑点 | 虚假盈利 |
| 数据泄漏 | 测试数据混入训练 | 不可重现 |

### 4.2 回测指标解读

**收益指标**：
```
总收益率 = (期末净值 - 1) × 100%
年化收益 = (期末净值^(250/交易日) - 1) × 100%
```

**风险指标**：
```
最大回撤 = max(峰值到当前的最大跌幅)
年化波动率 = 日收益标准差 × √252
下行波动率 = 只计算负收益的波动
```

**风险调整收益**：
```
夏普比率 = (年化收益 - 无风险利率) / 年化波动率
         └─ 每单位风险获得的超额收益

索提诺比率 = (年化收益 - 无风险利率) / 下行波动率
          └─ 只考虑下行风险的收益风险比

卡尔玛比率 = 年化收益 / 最大回撤
          └─ 每单位回撤获得的收益
```

**指标经验值**：
- 夏普 > 1：不错的策略
- 夏普 > 2：优秀的策略
- 最大回撤 < 15%：可接受
- 最大回撤 < 10%：优秀

### 4.3 样本外测试

**时间划分**：
```
|-------|-------|-------|
 训练集  验证集   测试集
```

**Walk-Forward分析**：
```
滚动窗口训练 → 验证 → 滚动向前
避免用全部数据一次性训练
```

**为什么重要**？
- 模拟真实交易环境
- 检验策略稳定性
- 防止过拟合

### 4.4 参数优化

**优化方法**：
- 网格搜索：穷举所有组合
- 遗传算法：进化式寻找最优
- 贝叶斯优化：智能选择下一组参数

**优化陷阱**：
- 过拟合：参数"记住"历史
- 数据窥探：反复测试同一数据
- 参数太多：维数灾难

**建议**：
- 参数越少越好（3-5个）
- 参数要有逻辑意义
- 做参数稳定性检验
- 样本外必须验证

### 4.5 风险管理体系

**仓位管理**：

```
凯利公式：f = (p×b - q) / b
├─ f: 投入资金比例
├─ p: 胜率
├─ q: 败率 = 1-p
└─ b: 盈亏比

等风险仓位：每笔风险相同
波动率目标：根据波动率调整仓位
```

**止损止盈**：

```
固定止损：亏损5%平仓
ATR止损：2倍ATR止损
时间止损：N天未动平仓
追踪止损：回撤N%平仓
```

**组合风控**：

```
├─ 单票仓位上限（如20%）
├─ 行业集中度限制（如30%）
├─ 总仓位控制（如80%）
└─ 相关性控制（同策略限制）
```

## 阶段五：实盘交易

### 核心目标

从模拟到实盘，建立稳定交易体系。

### 5.1 模拟盘阶段

**为什么必须模拟盘**：
- 验证策略在"实时"环境表现
- 熟悉交易系统操作
- 测试风控逻辑
- 建立交易纪律

**模拟盘关注**：
- 策略是否正常运行
- 数据是否及时
- 成交是否符合预期
- 异常情况处理

### 5.2 实盘对接

**交易接口类型**：

| 类型 | 说明 | 适合 |
|------|------|------|
| 模拟盘 | 模拟交易 | 学习阶段 |
| 券商API | 直接对接券商 | 正式实盘 |
| CTP接口 | 期货专用 | 期货交易 |
|交易所API | 加密货币 | 数字货币 |

**实盘注意事项**：
- 网络稳定性
- 异常处理
- 资金安全
- 数据延迟

### 5.3 交易流程

**盘前**：
```
1. 检查数据更新
2. 检查策略状态
3. 检查风控参数
4. 准备监控列表
```

**盘中**：
```
1. 策略自动运行
2. 人工监控异常
3. 记录异常情况
4. 必要时手动干预
```

**盘后**：
```
1. 导出交易记录
2. 计算当日收益
3. 分析异常交易
4. 准备明日计划
```

### 5.4 进阶方向

**高频交易**：
- 需要极低延迟
- 竞价市微观结构
- 专业设备

**期权策略**：
- 波动率交易
- Delta对冲
- 希腊值管理

**基本面量化**：
- 财务因子
- 事件驱动
- ESG因子

### 5.5 学习检查点

✅ 完成标志：
- [ ] 策略上模拟盘
- [ ] 建立复盘习惯
- [ ] 有风控体系
- [ ] 形成交易哲学

## 附录：关键概念速查

### A. 常用术语

| 术语 | 含义 |
|------|------|
| 多头/做空 | 看涨买入 / 看跌卖出 |
| 仓位 | 持有资产的价值比例 |
| 滑点 | 期望价与实际价的差异 |
| 回撤 | 从高点到低点的跌幅 |
| 夏普比率 | 风险调整后收益指标 |
| Alpha | 超额收益 |
| Beta | 相对市场的波动 |
| IC | 因子与收益的相关系数 |

### B. 常用指标公式速查

```
SMA = sum(close, n) / n
EMA = α × close + (1-α) × EMA_prev, α=2/(n+1)

RSI = 100 - 100/(1 + 平均涨幅/平均跌幅)

MACD = EMA(12) - EMA(26)
Signal = EMA(MACD, 9)

Boll_upper = MA(20) + 2×STD(20)
Boll_lower = MA(20) - 2×STD(20)

ATR = MA(TrueRange, 14)
TrueRange = max(H-L, │H-C_prev│, │L-C_prev│)
```

### C. 回测指标计算

```
总收益率 = (期末净值 - 1) × 100%
年化收益 = (期末净值^(250/交易日) - 1) × 100%
最大回撤 = max((峰值 - 当前值) / 峰值)
夏普比率 = (年化收益 - 无风险利率) / 年化波动率
胜率 = 盈利次数 / 总交易次数
盈亏比 = 平均盈利 / 平均亏损
```

### D. A股交易规则要点

```
交易时间： 9:30-11:30, 13:00-15:00
交易制度： T+1
最小单位： 100股(1手)
涨跌限制： ±10%(主板), ±20%(科创/创业)
成交原则： 价格优先、时间优先
交易费用： 印花税0.1%(卖出) + 佣金万2.5-万5
```

## 总结建议

### 给算法工程师的建议

1. **发挥你的优势**
   - ML/RL直接应用于策略
   - 特征工程用于因子挖掘
   - 优化算法用于参数搜索
   - 统计方法用于风控

2. **注意量化特点**
   - 数据信噪比低
   - 过拟合风险高
   - 市场环境变化快
   - 交易成本敏感

3. **从简单开始**
   - 先跑通一个简单策略
   - 理解每个环节
   - 逐步增加复杂度
   - 持续迭代优化

### 学习节奏建议

```
第1-2月：  基础知识 + 数据处理
第3-4月：  策略开发 + 回测
第5-6月：  风控优化 + 模拟盘
第7月+：   实盘交易 + 持续学习
```

### 心态建议

- 量化是一场长跑，不是一夜暴富
- 保住本金比赚钱重要
- 策略失效是常态，需要持续进化
- 纪律比策略更重要
